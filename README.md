# 🎧 Chinook Console Application (C# + SQLite + Dapper)

This project is a C# console application that connects to the Chinook music database using **Dapper ORM** and **SQLite**. It allows users to view customer and invoice data, as well as insert new customers directly from the console.

> 🔒 This is a personal educational project built to demonstrate database access, console I/O, and lightweight ORM use in C#.

---

## 🧠 Features

- ✅ Insert new customers via console input
- ✅ Retrieve and display customer records (ID, name, country, email)
- ✅ Retrieve and display invoice records (ID, customer, date, total)
- ✅ Lightweight SQL queries using Dapper ORM
- ✅ Connects to a local SQLite database
- ✅ Error handling and clean console formatting

---

## 💻 Technologies Used

- C# (.NET)
- SQLite
- Dapper ORM
- Console UI
- Visual Studio / Visual Studio Code
